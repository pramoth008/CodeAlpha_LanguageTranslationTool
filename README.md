# Language Translation Tool

## Description

A simple Language Translation Tool built using Python, Streamlit, and Deep Translator. The application allows users to translate text between multiple languages through a user-friendly interface.

## Features

* Enter text for translation
* Select source language
* Select target language
* Display translated text instantly
* Simple and interactive web interface

## Technologies Used

* Python
* Streamlit
* Deep Translator

## Installation

1. Clone the repository
2. Install the required packages:

pip install -r requirements.txt

3. Run the application:

streamlit run app.py
